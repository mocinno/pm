# Qv2ray-Win

## 1、下载地址

点击下载 👉[<mark style="color:orange;">Qv2ray</mark>](https://wwi.lanzout.com/ibDNexoxxbe)

## 2、配置订阅

### 1、使用压缩工具解压下载得到的 Qv2ray-v2.7.0-Windows.7z压缩包。

### 2、打开qv2ray.exe主程序文件。

{% hint style="info" %}
:woman\_mage:**Tips:** 若是电脑安装有360之类安全软件，弹框提示请选择**允许程序全部操作**。
{% endhint %}
